# CMD 命令提示符

***1、cd命令***

```text
//进入d盘
D:
//进入F盘
F:

cd /?     //获取使用帮助

cd \       //跳转到硬盘的根目录

cd C:\WINDOWS  //跳转到当前硬盘的其他文件

d:        //跳转到其他硬盘

cd /d e:\software    //跳转到其他硬盘的其他文件夹，注意此处必须加/d参数。否则无法跳转。

cd..      //跳转到上一层目录


//创建目录
md 目录名（文件夹）
//删除目录
rd 目录名（文件夹）

查看本机ip
ipconfig

清除屏幕
cls

复制文件
copy 路径\文件名 路径\文件名 ：把一个文件拷贝到另一个地方。

移动文件
move 路径\文件名 路径\文件名 ：把一个文件移动（就是剪切+复制）到另一个地方。

删除文件
del 文件名   //这个是专门删除文件的，不能删除文件夹

//用来测试网络是否畅通
ping ip(主机名)

```
